# Language Learning App
This project is a simple demonstration of creating a language learning application using React Native, Expo, TypeScript, Styled Components, and the Context API.

## Getting Started
To get started with this project, follow these steps:

1. **Clone the Repository:**
```
git clone https://github.com/danieltgfischer/leanguage-learning-app.git
```
2. **Install Dependencies:**
``` 
cd leanguage-learning-app
npm install
```
3. **Start the Development Server:**
```
npm start
````
4. **Download the Expo Go app:**
   - For iOS users, download the Expo Go app from the [App Store](https://apps.apple.com/us/app/expo-go/id982107779).
   - For Android users, download it from [Google Play](https://play.google.com/store/apps/details?id=host.exp.exponent&hl=en&gl=US).
5. **Scan the QR code:** 

   - After starting the development server using `npm start`, a QR code will be displayed in your terminal.
   - Open the Expo Go app on your mobile device and scan the QR code to load the app.
